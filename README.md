# GLP-1 léky na hubnutí — co říkají studie

Webová stránka shrnující data z **80+ klinických studií** o třech GLP-1 lécích na hubnutí: **Mounjaro** (tirzepatid), **Wegovy** (semaglutid) a **Saxenda** (liraglutid).

**→ [Zobrazit stránku](https://ronaldluc.github.io/glp1-leky/)**

## Co stránka obsahuje

- Jak GLP-1 léky fungují (srozumitelně, bez odborného žargonu)
- Porovnání tří léků (mechanismus, dávkování)
- Trávicí potíže s frekvencemi, očištěné o placebo, se srovnáním s běžným antibiotikem
- Pozitivní účinky: úbytek hmotnosti, snížení úmrtnosti, ochrana srdce/ledvin, prevence cukrovky
- 5 obav vyvrácených na milionech pacientů (rakovina štítné žlázy, sebevražedné myšlenky, pankreatitida, rakovina slinivky, úbytek svalů)
- Vliv silového cvičení na zachování svalové hmoty
